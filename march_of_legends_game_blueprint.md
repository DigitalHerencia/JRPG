# MARCH OF LEGENDS: FIRST CHAIR

## Concept Overview
A 2D pixel‑art JRPG parody set in a university marching band where musical formations function as magical combat abilities. The tone blends melodramatic JRPG storytelling with absurd self‑aware humor about band culture.

Inspirations:
- Final Fantasy VI
- Persona series
- EarthBound
- NCAA marching band halftime culture

Core pillars:
1. Tactical band formations as gameplay
2. Rhythm‑based solo battles
3. Over‑the‑top anime melodrama
4. Campus exploration and character progression

---

# World Setting

The story takes place at **Harmonia State University**, a desert campus famous for its legendary marching ensemble known as:

**"The Most Dangerous Marching Band in the World"**

Within the game's lore, musical formations performed on the field are actually ancient sigils capable of channeling musical energy into reality‑bending effects.

Halftime shows are effectively magical rituals.

# Gameplay Modes

## Mode 1 — Field Command

Top‑down tactical system where the player directs band formations.

Players draw shapes on the marching field grid.

Examples:

Circle formation → healing field  
Arrow formation → attack buff  
Dragon formation → boss summon attack


## Mode 2 — Rhythm Battles

During solos and duels the game switches to rhythm gameplay.

Inspired by:

- Guitar Hero
- Dance Dance Revolution

Inputs trigger musical attacks.

Perfect combos unlock **Improvisation Mode**.

Improvisation Mode allows freestyle patterns that generate powerful abilities.

# Strong Prompt Pack

## Prompt 1 — Master Game Architecture

"Design a full architecture for a 2D pixel JRPG in Godot where marching band formations act as tactical abilities and rhythm gameplay controls musical battles. Provide folder structure, scenes, systems, and modular scripts suitable for AI‑generated development."


## Prompt 2 — Formation Spell System

"Write a Godot system where player‑drawn formations on a grid are analyzed and converted into gameplay abilities. Detect shapes such as circles, arrows, spirals, and custom patterns and map them to abilities."


## Prompt 3 — Rhythm Combat Engine

"Create a modular rhythm gameplay system in Godot similar to Guitar Hero that supports scrolling notes, combo scoring, improvisation mode, and ability triggers based on performance accuracy."


## Prompt 4 — Tactical Band Movement

"Generate a tactical grid movement system for marching band units where multiple characters move in synchronized patterns across a football field grid."


## Prompt 5 — Procedural Halftime Shows

"Create a procedural generator that builds marching band formations and halftime show sequences for levels using predefined pattern rules."


## Prompt 6 — Pixel Art Prompt Generator

"Generate a library of prompts for producing SNES‑style pixel sprites for marching band characters, instruments, stadium environments, mascot characters, and enemy bands."
