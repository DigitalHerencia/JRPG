# Formation System Prompt

Design and implement the first real formation system.

## Goal
Replace the placeholder classification with a robust grid-based shape recognizer that can identify:
- line
- column
- box
- arrow
- V-shape
- spiral
- chaos/jazz fallback

## Requirements
- Godot 4 GDScript
- deterministic evaluation
- easy-to-tune thresholds
- debug overlay support
- clean API for ability resolution

Return code and test cases.
