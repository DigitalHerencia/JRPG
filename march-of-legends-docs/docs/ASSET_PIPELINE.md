# Asset Pipeline

## Rule zero
Do not block gameplay on art.

## Asset classes

### Environment
- campus tileset
- football field
- rehearsal hall
- stadium concourse
- dorm/exterior

### Characters
- Leo idle/walk sprites
- mascot idle/emote sprites
- band section archetypes
- rival conductors
- boss portraits

### UI
- note icons
- formation icons
- menu frames
- dialogue windows
- timing bar assets

### Effects
- brass blast
- metronome pulse
- crowd hype burst
- spotlight beam
- formation summon flashes

## Recommended sourcing mix

### Free/open assets
Use free packs for:
- terrain
- props
- generic UI pieces
- generic VFX bases

### AI-generated concept art
Use image generation for:
- portraits
- mascot concepts
- uniform variants
- rival band key art

### AI-generated pixel reference, then cleanup
Ask for sprite-sheet style references, but expect to refine or manually split them.

## Prompt pattern for image generation

Use this formula:

`[subject], SNES-inspired pixel art, readable silhouette, top-down RPG sprite sheet aesthetic, 16-bit color discipline, marching band uniform, desert university vibe, expressive pose, clean outline, no text, transparent background`

## Critical warning
A lot of AI "pixel art" is fake pixel art: it is painted to look pixel-ish but does not align to a usable grid. Treat AI output as concept/reference unless it lands cleanly enough to edit into a proper sprite sheet.
