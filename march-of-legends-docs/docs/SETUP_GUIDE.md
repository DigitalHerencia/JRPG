# Setup Guide

## Install

1. Install Godot 4.x standard build.
2. Install VS Code.
3. Install Git.
4. Optional: install GitHub Desktop if terminal Git makes you feel like Victorian dentistry.

## Open the project

1. Launch Godot.
2. Import the `godot/` folder.
3. Open `project.godot`.
4. Run the main scene.

## First milestone

You are done with setup when all of these are true:

- main menu opens
- you can enter campus scene
- you can enter field command scene
- you can enter rhythm battle scene
- placeholder controls respond

## The only acceptable first-week goal

One ugly but playable vertical slice.
