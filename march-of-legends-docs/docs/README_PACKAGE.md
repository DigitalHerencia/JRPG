# March of Legends: First Chair — Starter Package

This package contains three things:

1. `godot/` — a Godot 4 starter scaffold for a 2D pixel-art JRPG/rhythm/tactical hybrid.
2. `notion_import/` — CSV files you can import into Notion as starter databases.
3. `docs/` — implementation docs, prompts, and setup notes.

## What this scaffold is optimized for

- first-time game dev in Godot
- heavy AI-assisted / vibe-coded development
- 2D only
- pixel-art pipeline with AI-generated concept + sprite refinement
- modular systems that can expand without rewriting the whole repo

## Recommended build target

- desktop first
- web export second

## Why this shape?

The scaffold is split around Godot scenes, scripts, data, and assets so an agent can work on one slice without spraying chaos through the repo.

The code is intentionally placeholder-heavy. It is meant to compile into a navigable skeleton, not a finished game.
